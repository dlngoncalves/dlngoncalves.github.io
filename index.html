Educa Money

Role: Lead Backend & Porting Engineer

Managed the technical infrastructure for an educational math and finance game, architecting the Node.js/MariaDB backend and leading the technical migration from PC to Mobile platforms..

Game Website App Store Google Play

My Role & Technical Focus

On EducaMoney, I focused on the platform infrastructure and the data pipeline required to turn a math-learning game for children into a viable classroom tool. My primary responsibility was the full-stack architecture, ensuring student data was handled securely via a cloud-hosted backend and leading the technical refactoring required to bring the desktop experience to mobile devices.

Beyond architecting the backend and leading the mobile port, I developed a Visual Analytics system that allowed teachers to monitor student progress in real-time. By leveraging Azure Data Explorer and KQL, I transformed raw gameplay data into actionable insights for educators—directly applying my academic research in Visual Analytics to a commercial product.

I also implemented the accessibility frameworks that allowed the game to be used by students with visual impairments.

  • Visual Analytics & KQL: Engineered a data-monitoring tool using Azure Data Explorer and KQL, enabling teachers to track learning milestones and measure pedagogical outcomes.
  • Backend Engineering: Developed and deployed the Node.js/MariaDB infrastructure on Azure and AWS to handle player progress and data synchronization.
  • Platform Migration: Refactored the core code to ensure stable performance and UI scaling during the port from PC to Mobile.
  • Accessibility Implementation: Engineered specialized UI and audio systems to make math and money concepts accessible to visually impaired students.
