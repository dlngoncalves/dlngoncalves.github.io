About me

I am a Senior Software Engineer and Game Developer currently based in Porto Alegre, Brazil. I have over 7.5 years of professional experience in the Games Industry, primarily as a Unity Developer. My career is defined by the strategic integration of a deep academic foundation in Visual Analytics and Computer Graphics with a proven track record of architecting high-performance commercial games and scalable backend systems.

  • Academic Foundation: I hold a Master’s degree in Computer Science with a focus on data-intensive fields, including Visual Analytics, Big Data, and Machine Learning. For my dissertation, I developed a specialized Visual Analytics tool designed to process and visualize large volumes of multivariate trajectory data.
    • This tool was built using C# and WPF, leveraging Azure Data Explorer for backend data processing and KQL for querying. The project not only demonstrated my ability to handle complex data visualization challenges but also provided insights into user interaction with large datasets, which has informed my approach to game development and user experience design.
  • Professional Impact: I have a proven track record of leading teams and delivering complex commercial projects. My work on Pets Rules involved developing custom designer tools and optimizing mobile build pipelines, while my contributions to PSG Football Freestyle - winner of the BIG Festival 2018's "Brands Award" - focused on core gameplay programming, PlayFab backend integration and custom designer tools. Finally, in the project Educa Money, I developed the entire backend infrastructure. For this project, I also developed an advanced Visual Analytics solution, using Azure Data Explorer and the KQL language, drawing insights from my experience studing Visual Analytics in academia.
